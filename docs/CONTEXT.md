## 2026-07-17 - 项目初始化

### 项目概述
创建一个网页收藏（书签导航）网站，1:1 复刻 https://nav.iowen.cn/bookmark 的功能和界面，并在此基础进行扩展和功能增强。

### 原始网站分析

**网站名称**：iowen 导航 / 新标签页
**技术栈**：WordPress + iotab 插件（自定义标签页插件）
**核心功能**：

1. **壁纸背景系统**
   - 随机壁纸展示（`/wp-content/uploads/wallpapers/images/`）
   - 模糊度调节（`--wallpaper-blur`）
   - 亮度调节（`--wallpaper-brightness`）

2. **实时时钟显示**
   - 时:分:秒 实时更新

3. **多引擎搜索框**
   - 支持切换搜索引擎（百度、Google 等）
   - 搜索智能提示（smart tips）
   - 快捷键操作（Enter 搜索，方向键导航）

4. **收藏图标网格**
   - 按分类展示网站图标
   - 图标支持拖拽排序（Sortable.js）
   - 图标大小、间距可配置
   - 分类标签：影音视听、站长工具、网盘云储、用户演示、UED团队、Chrome插件、在线工具等

5. **添加图标弹窗（Modal）**
   - 从分类库中选择已有图标
   - 自定义添加图标（4种模式）：
     - 在线图标：自动从链接获取 favicon
     - 上传图标：本地上传图片
     - 纯色图标：纯色背景 + 文字
     - 链接图标：以链接作为图标
   - 颜色选择器（预设色板 + 自定义颜色）

6. **用户系统**
   - 登录后可同步配置到云端
   - 未登录提示登录体验完整功能

7. **小组件（Gadget）系统**
   - 添加小组件功能（待进一步分析）

### 技术特征
- 基于 WordPress REST API（`/wp-json/io/v1/bm/`）进行数据交互
- 前端使用 jQuery + 自定义 CSS（bm.css）
- 图标使用 iconfont 字体图标
- 图片裁剪使用 cropper.js
- 拖拽排序使用 Sortable.js
- 支持 HMTL 静态缓存（isCacheHtml）
- 图标数据支持加密传输（isEncrypted）

### 项目目标
1. 完全 1:1 复刻原始网站的所有前端界面和交互功能
2. 不依赖 WordPress 后端，使用纯前端 + 轻量后端存储方案
3. 为后续功能扩展提供良好的架构基础
